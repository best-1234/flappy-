# 🐦 Flappy Bird (Beginner OOP Project)

## 📌 About
This is a simple Flappy Bird game built using Python and Pygame.

This project is part of my learning journey where I am practicing Object-Oriented Programming (OOP) for the first time.

## 🎯 Purpose
- Learn OOP concepts (classes, objects)
- Understand game logic using Pygame
- Practice real project building as a beginner

## 🎮 Features
- Basic bird movement with gravity
- Pipe obstacles
- Collision detection
- Score system

## ⚠️ Limitations
- No restart feature yet
- Basic graphics and mechanics
- Still improving as I learn

## 🛠️ Tech Used
- Python
- Pygame

## ▶️ How to Run
1. Install pygame:
   pip install pygame

2. Run the game:
   python flappy.py

## 📚 What I Learned
- Creating classes (Bird, Pipe)
- Game loop logic
- Collision detection basics
- Managing objects in a game
